Some Comments

This is the beta launch set of games for the launch of Eclectic Puzzles. There are another 10 or so planned. You might say in development, but as this is a one man show, they won't be in development until this is finished. Also, each of these first five is actually the prototype of a family of similar games of increasing complexity. Right now these are only available on the internet and not really suitable to the small screens. Learning how to build phone apps is on the queue, though the queue is pretty full right now. Retirement means picking up new hobbies.

These are games but also puzzles. They are designed using mathematical principles. It is best to develop techniques to solve the puzzles, but for each new game the solution is different. A good example is Sudoku. Getting to the solution requires a set of strategies but each game played those strategies need to be used to reach the solution for that game. These first five are simple and can be played naively, but future additions will be more complex.
